📊 Madhav E-commerce Sales Dashboard
📌 Project Overview

This project is a Power BI dashboard created to analyze and visualize e-commerce sales data. The goal of this project is to convert raw business data into meaningful insights that help in better decision-making.

Through this dashboard, I focused on understanding sales performance, profit trends, customer behavior, and category-wise distribution in a simple and interactive way.

🎯 Objective

The main objective of this project is to:
Analyze sales and profit performance
Identify top-performing states and customers
Understand category-wise sales distribution
Track monthly profit trends
Help businesses make data-driven decisions

🛠️ Tools & Technologies Used

Power BI
Data Cleaning & Transformation
Data Visualization
Basic Business Analytics
